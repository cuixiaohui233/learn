tuple元祖

tuple初始化就不能修改是不可变的，没有append和insert这些方法。
定义：
t = (1,2)
如果是一个空的：
t = ()
如果是一个元素：
t = (a,)
“可变的tuble”:
t = ('a','b',['c','d'],'e')
要想拿到'c':
t[2][0] = '哈哈哈'
再次输出t:
('a','b',['哈哈哈','d'],'e')
以上可以得出结论：
tuple的不可变是说tuble的每个元素指向不可变，即指向’a‘,就不可以指向’b‘,指向一个list们就不可以指向别的类型，但是！这个list本身是可变的。
